Pipeline Examples

Two parallel examples: Declarative and Scripted. Use whichever fits your pipeline style.

Declarative (parallel stages)

pipeline {
  agent any

  stages {
    stage('Parallel Tests') {
      parallel {
        stage('test') {
          steps {
            echo 'Running test.py'
            sh 'python3 test.py'
          }
        }
        stage('test-2') {
          steps {
            echo 'Running test-2.py'
            sh 'python3 test-2.py'
          }
        }
        stage('test-3') {
          steps {
            echo 'Running test-3.py'
            sh 'python3 test-3.py'
          }
        }
      }
    }
  }
}

Scripted (parallel map)

node {
  stage('Parallel Tests') {
    def branches = [:]
    branches['test'] = {
      stage('test') { sh 'python3 test.py' }
    }
    branches['test-2'] = {
      stage('test-2') { sh 'python3 test-2.py' }
    }
    branches['test-3'] = {
      stage('test-3') { sh 'python3 test-3.py' }
    }
    parallel branches
  }
}

Place the repository `Jenkinsfile` (Declarative) at the project root and create a Multibranch Pipeline or a Pipeline job that points to this repo.
